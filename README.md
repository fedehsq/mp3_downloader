# Mp3 Dowloader #
Download mp3 directly to your Android or Windows device.

## Android ##
Download and install 'app-release.apk'  from 'Android' directory.

## Windows ##
Download the 'mp3_downloader_windows' folder inside 'Windows' and run 'mp3_downloader.exe'.
